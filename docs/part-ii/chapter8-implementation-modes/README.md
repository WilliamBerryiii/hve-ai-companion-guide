## Learning Objectives

By the end of this chapter, you will:

* Execute implementation plans using Inline Chat for precise file modifications
* Create new files and scaffolding using the `/new` command and Agent mode
* Accelerate line-level coding with Inline Suggestions
* Apply test-driven implementation workflow (write tests first, implement to pass)
* Select the optimal tool for each implementation task
* Verify each implementation step using plan-defined criteria
* Recognize when implementation diverges from plan and adapt appropriately
