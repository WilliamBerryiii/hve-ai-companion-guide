Now you'll transform those plans into working code using three powerful implementation modes: Edit Mode for precise modifications, Insert Mode for scaffolding new files, and Inline Copilot for rapid line-level coding.

## Learning Objectives

By the end of this chapter, you will:

* Execute implementation plans using Edit Mode for precise file modifications
* Create new files and scaffolding using Insert Mode with template patterns
* Accelerate line-level coding with Inline Copilot suggestions
* Apply test-driven implementation workflow (write tests first, implement to pass)
* Select the optimal mode for each implementation task
* Verify each implementation step using plan-defined criteria
* Recognize when implementation diverges from plan and adapt appropriately
