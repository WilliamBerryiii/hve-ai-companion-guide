## What you'll learn

By the end of this chapter, you will:

* Conduct deep research using Task Researcher mode for typically 20-60 minute investigations, depending on complexity
* Create comprehensive research documents with evidence citations and line references
* Apply evidence gathering techniques across internal codebases and external sources
* Integrate MCP tools for SDK documentation and authoritative external research
* Recognize when research is "complete" and ready for planning
* Apply the D-RPI variation for unfamiliar brownfield codebases
