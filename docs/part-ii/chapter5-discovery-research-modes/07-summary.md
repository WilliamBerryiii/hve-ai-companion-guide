## Key Takeaways

### Ask Mode is reconnaissance, not implementation

* 5-10 minute discovery sessions build context rapidly without commitment
* Landscape â†’ Filter â†’ Exemplar â†’ Convention â†’ Decision pattern provides structure
* Lightweight notes capture findings without derailing momentum

### Question quality determines answer quality

* Broad questions establish landscape, specific questions get actionable details
* Context provision (project goals, constraints, attempted solutions) grounds answers in reality
* @workspace directive anchors Copilot to codebase, preventing generic advice

### Incomplete answers have recognizable patterns

* Generic advice signals insufficient context in question
* "It depends" without specifics signals need for disambiguation
* Missing citations signal shallow scanning versus deep analysis
* Probing techniques (request examples, ask comparisons, validate with files) verify answer quality

### Escalation decisions at 10 minutes

* Sufficient for implementation: Pattern clear, exemplar found, conventions understood
* Sufficient for focused research: Pattern identified but needs comprehensive documentation
* Insufficient: Multiple patterns unclear, contradictions, significant gaps

### Same-thread workflow multiplies efficiency

* Ask Mode loads 15-20 files into context during discovery
* Task Researcher starts "warm" instead of cold
* Research quality improves when context carries forward
